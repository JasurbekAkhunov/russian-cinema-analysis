# Анализ российского кинопроката

## Постановка задачи

Заказчик — **Министерство культуры Российской Федерации**. Цель — изучить рынок российского кинопроката, выявить текущие тренды и оценить, насколько фильмы, получившие государственную поддержку, интересны зрителям.

Данные опубликованы на портале открытых данных Министерства культуры и содержат информацию о прокатных удостоверениях, кассовых сборах, государственной поддержке, а также рейтинги с КиноПоиска.

## Данные

| Файл | Описание |
|------|----------|
| `mkrf_movies.csv` | Реестр прокатных удостоверений: название, дата премьеры, тип, режиссёр, жанр, рейтинг, бюджет, господдержка |
| `mkrf_shows.csv` | Сборы в российских кинотеатрах по номеру прокатного удостоверения |

### Ключевые поля `mkrf_movies.csv`

| Столбец | Описание |
|---------|----------|
| `title` | Название фильма |
| `puNumber` | Номер прокатного удостоверения |
| `show_start_date` | Дата премьеры |
| `type` | Тип фильма |
| `director` | Режиссёр |
| `age_restriction` | Возрастная категория |
| `refundable_support` | Возвратная господдержка (руб.) |
| `nonrefundable_support` | Невозвратная господдержка (руб.) |
| `budget` | Общий бюджет (руб.) |
| `ratings` | Рейтинг на КиноПоиске |
| `genres` | Жанры |

## Этапы работы

### 1. Предобработка данных
- Объединение таблиц `mkrf_movies` и `mkrf_shows` по `puNumber` (left join)
- Преобразование типов: `show_start_date` → datetime, числовые столбцы → float
- Обработка пропусков: господдержка → 0 (нет поддержки), рейтинг → оставлен как NaN
- Исправление дубликатов в категориальных столбцах (регистр, пробелы)
- Исправление значений поля `type`

### 2. Новые признаки
- `year` — год выхода фильма
- `main_director` — имя первого режиссёра из списка
- `main_genre` — первый жанр из списка
- `state_support_share` — доля господдержки от бюджета

### 3. Ключевые результаты

**Динамика проката**: наиболее полно представлен период 2015–2019. Доля фильмов с данными о кинопрокате растёт год от года.

**Сборы по годам**: максимальная сумма сборов зафиксирована в 2019 году, минимальная — в ранние периоды. Среднее значение сборов существенно выше медианного из-за нескольких блокбастеров.

**Возрастные ограничения**: фильмы с рейтингом **6+** и **12+** собирают наибольшую кассу — более широкая аудитория.

**Государственная поддержка**: поддержанные фильмы имеют более высокий средний рейтинг, однако ROI (отношение сборов к бюджету) у большинства ниже 1 — сборы не окупают затраты.
